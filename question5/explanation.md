# Question 5: Integration

In real world systems like a temperature-controlled thermostat, control flow, functions, and embedded systems concepts work together to make the system operate automatically. The sensor collects data and sends it to the controller. Functions organize tasks such as reading the sensor, processing data/values, and controlling the LED. Control flow structures like loops and if statements determine when these functions run and what actions the system should take. Together, these concepts allow the system to continuously monitor temperature and turn the LED on when the temperature exceeds or is below a set threshold.
